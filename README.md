Змейка на C# 🐍
Консольная игра "Змейка", созданная в учебных целях для изучения Git и GitHub.
Как играть 🎮
- Стрелки управления: ⬆️ ⬇️ ⬅️ ➡️
- Еда: символ `@`
- Змейка: символ `O`
- Цель: съесть как можно больше еды, не врезавшись в стены и не наступив на свой хвост
Установка и запуск ⚙️
1. Убедитесь, что установлен .NET SDK (https://dotnet.microsoft.com/download)
2. Клонируйте репозиторий:
git clone https://github.com/ВАШ-ЛОГИН/SnakeGame.git
3. Перейдите в папку проекта:
cd SnakeGame
4. Запустите игру:
dotnet run
Этапы разработки 📝
1. Движущаяся голова - базовая механика и границы
2. Хвост змейки - добавлен список сегментов
3. Еда и рост - генерация еды, увеличение длины
4. Game Over и очки - условия проигрыша, подсчет очков
Технологии 💻
- C# (.NET Console Application)
- Git / GitHub
